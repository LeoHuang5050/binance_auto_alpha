# Binance Auto Trade - 币安量化交易系统

一个基于Python的币安量化交易系统，提供图形化界面来监控代币价格。

## 功能特性

- 🚀 **实时价格监控**: 使用币安官方API获取代币最新价格
- 💰 **多代币支持**: 同时监控多个代币的价格变化
- 📊 **24小时变化**: 显示代币24小时价格变化百分比
- 🎨 **图形化界面**: 基于tkinter的现代化GUI界面
- ⚡ **自动更新**: 每30秒自动更新价格数据
- 📝 **系统日志**: 实时显示操作日志和错误信息
- 🖱️ **右键菜单**: 支持删除和刷新单个代币

## 安装要求

- Python 3.7+
- tkinter (通常随Python一起安装)
- requests库

## 快速开始

### 方法一：直接运行Python脚本
```bash
# 安装依赖
pip install -r requirements.txt

# 运行程序
python binance_trader.py
```

### 方法二：打包成exe文件（推荐）
```bash
# 使用自动打包脚本
python build_exe.py

# 或使用批处理文件
build.bat
```

打包完成后，在 `dist` 文件夹中找到 `BinanceAutoTrade.exe` 文件，双击即可运行。

## 使用方法

### 添加代币
1. 在"代币名称"输入框中输入代币符号（如：BTCUSDT、ETHUSDT）
2. 点击"添加代币"按钮或按回车键
3. 系统会自动获取代币的最新价格和24小时变化

### 监控代币
1. 添加代币后，点击"开始监控"按钮
2. 系统会每30秒自动更新所有代币的价格
3. 在表格中查看实时价格和变化情况

### 管理代币
- **删除代币**: 右键点击表格中的代币，选择"删除代币"
- **刷新价格**: 右键点击表格中的代币，选择"刷新价格"
- **清空列表**: 点击"清空列表"按钮删除所有代币

## 界面说明

### 主界面组件
- **代币输入框**: 输入要监控的代币名称
- **代币列表表格**: 显示代币名称、最新价格、24h变化、更新时间
- **控制按钮**: 开始/停止监控、清空列表
- **系统日志**: 显示操作记录和错误信息
- **状态标签**: 显示当前系统状态

### 支持的代币格式
- 现货交易对：BTCUSDT、ETHUSDT、BNBUSDT等
- 期货交易对：BTCUSDT、ETHUSDT等
- 确保代币名称格式正确（通常为大写字母+USDT）

## API说明

本应用使用币安公开API：
- 价格接口：`GET /api/v3/ticker/price`
- 24小时统计：`GET /api/v3/ticker/24hr`

## 技术栈

- **Python 3.7+**: 主要编程语言
- **tkinter**: GUI界面框架
- **requests**: HTTP请求库
- **threading**: 多线程处理
- **json**: 数据解析

## 常见问题
